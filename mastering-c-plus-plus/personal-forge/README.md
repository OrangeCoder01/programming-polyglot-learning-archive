# Personal Forge Folder

The **personal-forge** folder for **C++** holds both **challenges** and **projects** completed outside any curriculum (courses, workshops, coding challenges from websites, etc.), **except** curricula generated by an AI model.

The **challenges** and **projects** are mostly created out of a personal desire to solve real-life problems or complete tasks/objectives given by an AI model such as **ChatGPT**, which serves as one of the best mentors for assessing and improving my understanding and skills — providing coding **challenges**, followed by **projects**, while discussing and pointing out flaws in my code.

## Architecture inside the `personal-forge` subfolders

Each folder is named to reflect its **order**, **purpose**, and **programming language**.

- **Order** — follows a chronological listing (the sequence in which the code was completed)
- **Purpose** — describes what you should expect from the code inside the subfolder
- **Language** — indicates the programming language used

**Format:** `(order)_(purpose)_(language)`

**Example:** `15_armstrong_number_cpp`

- `15` — the 15th piece of code completed. Earlier code reflects my earlier coding style; later code showcases a more professional approach.
- `armstrong_number` — indicates the code produces output related to an Armstrong Number.
- `cpp` — written in C++.
