# ${\color{blue}\text{Fake Binary:}}$
## ${\color{orange}\text{Concept Explanantion:}}$
This is a codewar challenge called "Fake Binary" <br>
which returns a string of binary number 1011110, where each 1 is when the user's inputs's digit more than or equal to 5 otherwise 0<br>
${\color{pink}\text{For example}}$: Input = 5489256 ----> "1011011"
