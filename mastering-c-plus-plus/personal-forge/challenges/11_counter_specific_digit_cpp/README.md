# ${\color{blue}\text{Counter Specific Digit:}}$
## ${\color{orange}\text{Concept Explanantion:}}$
${\color{blue}\text{Counter Specific Digit:}}$ is about counting the number of digits the user asked to count within the user's validated input.<br>
It uses modulus and integer division for separating the digits and comparing the digit with specific digit, if they are equal, then the count increases
